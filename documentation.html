Nourt

Theme documentation

Welcome to Nourt, a clean, modern Shopify theme. This guide covers setup, customization, and the theme's built-in features. It is written for merchants — no coding required for anything described here.

Before you edit theme code: always duplicate your theme first (Online Store → Themes → ⋯ → Duplicate) and edit the copy. Custom code changes are not covered by theme support and can be overwritten by theme updates. Test on the duplicate, then publish once you're happy.

1. Getting started

  1. Add the theme to your store and click Customize to open the theme editor.
  2. Set your logo and favicon: theme editor → Header section (logo) and Theme settings → Logo (favicon).
  3. Build your home page from sections (see §3).
  4. Configure colors and fonts under Theme settings (see §4).
  5. Set up your menus: Online Store → Navigation → main-menu and footer.
  6. Preview on mobile and desktop, then Publish.

2. Home page & templates

Nourt uses Online Store 2.0 JSON templates, so you can add, remove, and reorder sections on every page from the theme editor — home, product, collection, cart, blog, article, search, page, and more.

To edit a template: theme editor → template dropdown (top center) → pick the page → add/reorder sections in the left panel.

3. Sections & blocks

Add a section with Add section in the theme editor. Highlights include:

  • Hero slideshow — full-width rotating banners (swipeable, autoplay optional).
  • Category circles / mosaic — shop-by-category navigation.
  • Collection grid / slider / 2-up — merchandise products and collections.
  • Product features / how it works / stats bar — storytelling blocks.
  • Bento banner, promo banners, lookbook — editorial layouts.
  • Testimonials, brand marquee, FAQ accordion — trust and support.
  • Newsletter, newsletter pop-up — capture signups.
  • Blog grid / hero / featured — content marketing.
  • Rich text, page content, contact form — flexible content pages.

Each section exposes settings (heading, layout, colors, content) in the theme editor. Product and featured-product sections also accept app blocks, so you can drop in compatible apps without code.

4. Theme settings

Open Theme settings (gear icon) in the editor:

  • Logo — logo image, logo width, favicon, and a text wordmark fallback.
  • Colors — color schemes with paired background/foreground values. Set at least your primary brand colors; sections can each pick a scheme.
  • Typography — heading font and body font (from the Shopify font library; the theme defaults to Poppins). Bold and italic variants load automatically.
  • Layout — container width, spacing, corner radius.
  • Commerce — free-shipping threshold, low-stock threshold, product rich snippets (SEO), predictive-search popular queries.
  • Social media — links for the footer/social icons (leave blank to hide).

5. Product pages

  • Media gallery with zoom, video, and 3D model support; variant images swap on selection.
  • Variant picker with color swatches, quantity selector, add-to-cart, and dynamic checkout (Shop Pay) buttons.
  • Shop Pay Installments banner, store pickup availability, related and complementary product recommendations, and a Follow on Shop button.
  • Reviews render from product metafields (custom.rating_avg, custom.rating_count, custom.reviews) — no app required.

6. Cart

  • Choose a cart drawer or cart page experience.
  • Line-item quantity updates, cart notes, discount codes, gift-card codes, free-shipping progress bar, and a tax-inclusive indicator where applicable.

7. Internationalization

Nourt ships with English, French, Arabic (full right-to-left), and Spanish, plus additional locale files. Add a country/currency and language selector via Shopify Markets; both appear in the header/footer when enabled.

8. Search

Predictive search (products, collections, pages, articles) with a full results page, filters, sorting, and a clear "no results" state.

9. Frequently asked questions

How do I change the logo?

Theme editor → Header → Logo image, and adjust Logo width. Upload a favicon under Theme settings → Logo.

How do I change fonts?

Theme settings → Typography → pick heading and body fonts from the Shopify font library. Bold/italic variants load automatically.

My menu isn't showing.

Menus come from Online Store → Navigation. Make sure the Header section's menu is set to main-menu (or your menu) and the footer to footer.

How do product reviews work?

Reviews use product metafields (no app needed). Populate custom.rating_avg, custom.rating_count, and custom.reviews on your products.

How do I show pickup availability?

Enable local pickup on your locations in Shopify admin; the product page shows availability automatically for eligible variants.

Can I use my own font?

The theme uses the Shopify font library for performance and reliability. Choose the closest match in Typography.

Something looks broken after I edited the code.

Revert to your duplicated backup (Themes → ⋯ → the copy you saved) and re-apply changes carefully. Custom code is outside the scope of theme support.
